# حاسبة أسعار التوريد والتركيب (السعودية)

تطبيق مخصص لحساب تكاليف "التوريد والتركيب" (Supply and Apply) للمقاولات والمشاريع في المملكة العربية السعودية. يتيح لك التطبيق إدخال بنود الأعمال، وحدات القياس، الكميات، سعر التوريد وسعر التركيب بالريال السعودي (SAR)، ويقوم بحساب الإجمالي لكل بند والمجموع النهائي للمشروع.

## المميزات
* **واجهة مرنة:** تصميم يتوافق مع أجهزة الجوال والشاشات الكبيرة بحيث يعرض جدولاً (Spreadsheet) على الشاشات الكبيرة وبطاقات على الجوال.
* **حساب فوري:** حساب الإجمالي تلقائياً لكل بند وللمشروع ككل عند تغيير أي رقم.
* **تصدير إلى إكسل (CSV):** إمكانية حفظ ومشاركة التسعيرة كملف إكسل بضغطة زر واحدة لإرسالها للعملاء أو الاحتفاظ بها كمسودة.

## التقنيات المستخدمة
* **Flutter:** لبناء واجهة المستخدم وتجربة التطبيق عبر منصات متعددة.
* **csv:** لإنشاء ملفات الإكسل (CSV).
* **share_plus:** لمشاركة الملفات وحفظها على الجهاز.

## طريقة التشغيل

1. تأكد من تثبيت [Flutter](https://docs.flutter.dev/get-started/install).
2. قم بتثبيت الحزم المطلوبة:
   ```bash
   flutter pub get
   ```
3. شغل التطبيق:
   ```bash
   flutter run
   ```

## ملاحظات حول التصدير (Export)
* عند الضغط على زر التنزيل في أعلى الشاشة، سيتم إنشاء ملف `pricing_sheet.csv`.
* على الجوال، ستظهر شاشة المشاركة (Share Sheet) لتتمكن من إرسال الملف عبر واتساب، الإيميل، أو حفظه في ملفات الجهاز.
* لفتح الملف بشكل صحيح في Microsoft Excel، يمكنك استخدام ميزة استيراد البيانات (Data > From Text/CSV) إذا واجهت أي مشاكل مع الترميز، رغم أن التطبيق يستخدم الصيغة القياسية.
